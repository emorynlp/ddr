Overview

The Deep Semantic Representations (DSR) project provides guidelines for the dependency relations reflecting deep semantics. The goal of this project is to represent sentences conveying the same semantics into one dependency structure, regardless of their surface forms. It is an ongoing project led by the Emory NLP research group.

Primary Dependencies

Primary dependencies are used to generate dependency trees preserving the following four properties: single root, single head, acyclic, and connected.

Secondary Dependencies

Secondary dependencies are used to generate dependency graphs containing dependency relations that are avoided from the primary dependencies to preserve the tree properties.
