Overview

Deep Semantic Representations (DSR) provide semantically-oriented dependency graphs reflecting deep structure. The goal of this project is to represent sentences conveying the same semantics into one unified dependency structure, regardless of their surface forms. This is an ongoing project led by the Emory NLP research group.

Dependency Relations

Primary dependencies (represented in solid lines above the words) are used to generate dependency trees preserving the following four properties: single root, single head, acyclic, and connected. Secondary dependencies (represented in dotted lines beneath the words) are used to generate dependency graphs consisting of dependency relations that are discarded from the primary dependencies to preserve the tree properties.
