oprd : object predicate

An object predicate (oprd) is the non-verbal predicate of the direct object in a small clause. We follow the flat analysis for small clauses such that the oprd relation stays the same for both active and passive constructions.


The object predicate "the baptist" in the small clause "John the baptist".

The object predicate "the baptist" in the passive construction.

The object predicate "the baptist" in the reduced passive construction.

The object predicate can be a non-nominal phrase.


The object predicate as the adjective phrase "smart".

The object predicate as the adjective phrase "smart" in the passive construction.

The object predicate as the adjective phrase "smart" in the reduced passive construction.

Similarly to copula, when a preposition phrase is the object predicate, the preposition becomes the head of the phrase.


The object predicate as the preposition phrase "after the baptist".
